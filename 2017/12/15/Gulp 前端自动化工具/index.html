Gulp.js

Gulp.js 是一个自动化构建工具,开发者可以使用它在项目开发过程中自动执行常见任务。

Gulp.js 是基于 Node.js 构建的，首先安装 Node.js 。

查看 node 版本

安装 Gulp.js

首先全局安装 Gulp

查看 Gulp 版本

在 Gulp 项目文件下局部安装 Gulp

配置package.json文件

Gulp 方法

在项目根目录下创建一个名为 gulpfile.js 的文件

作用：布置一些任务，通过 gulp 实现任务的调度

task()

运行 gulp

src(),dest(),pipe()

src([]) 源文件路径
dest() 目标文件路径 (可以自动创建目录)
pipe() 管道方法 表示输送(下一步)

单独执行特定的任务（task）

watch()

watch(监听的文件,[任务名]) 监听
